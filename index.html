JavaScript Lletres Repetides

Exercici 1

  1. Crea un array amb el teu nom on cada posició correspongui a una lletra.
  2. Fes un bucle que recorri aquest array i mostri per consola cadascuna de les lletres.

Exercici 2

  1. Al bucle, si la lletra és una vocal imprimeix a la consola: ‘He trobat la VOCAL: __’. Sinó, imprimeix: ‘He trobat la CONSONTANT: __’.
  2. Si trobes un número, mostra per pantalla: ‘Els noms de persones no contenen el número: __’

Exercici 3

  1. Emmagatzemar en array les lletres i nombre de vegades que apareixen. Utilitza el mètode map().

Exercici 4

  1. Crea una altra array amb el teu cognom on cada posició correspongui a una lletra.
  2. Fusiona els dos arrays en un. A més, afegeix una posició amb un espai buit entre la primera i la segona. És a dir, partim de 'array name i surname i al acabar l’execució només tindrem una que es dirà fullName.
